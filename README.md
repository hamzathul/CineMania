## Run Locally

Clone the project

```bash
  git clone https://github.com/hamzathul/netflix-clone.git
```

Go to the project directory

```bash
  cd netflix-clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

- Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
